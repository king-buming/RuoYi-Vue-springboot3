# 手机端（工人端 H5）启动说明

## 访问方式

手机端是纯静态 HTML 页面，无需编译，前端 dev server 启动后直接访问。

## 同伴启动步骤

### 1. 确保后端已启动
```bash
cd RuoYi-Vue-springboot3
mvn clean install -DskipTests
nohup java -jar ruoyi-admin/target/ruoyi-admin.jar > backend.log 2>&1 &
# 验证：curl http://localhost:8080 应返回 200
```

### 2. 启动前端 dev server
```bash
cd ruoyi-ui
npm install        # 首次需要
npm run dev        # 默认 80 端口
# 如果 80 端口被占，用：port=8081 npm run dev
```

### 3. 浏览器打开
```
http://localhost:8081/mobile.html     （如果 80 被占）
http://localhost/mobile.html          （如果 80 可用）
```

## 手机端登录方式

- 先在后台管理（admin/admin123 登录）→ 人员管理 → 人员档案 → 新增一个人员
- 记住该人员的：手机号 + 身份证后6位
- 手机端用这个手机号 + 身份证后6位登录

## 手机端功能

| 页面 | 功能 |
|------|------|
| 登录 | 手机号 + 身份证后6位 |
| 打卡 | 签到 / 签退（含拍照上传） |
| 记录 | 打卡历史 |
| 我的 | 个人资料、人脸状态、身份证上传（人像面+国徽面）、资质上传 |

## 后台审核流程

1. 后台 → 审核记录 → 新增 → 选人员 → 看人脸+身份证+资质 → 通过/驳回
2. 人员审核通过时，会自动检查该人员所有角色的资质要求
3. 缺证件会弹黄色警告，证件齐全则人员+所有证件同步通过

## 文件位置

```
ruoyi-ui/public/mobile.html    ← 手机端前端（单文件）
ruoyi-admin/.../controller/app/   ← 手机端后端接口（/app/*）
```

## 后端接口白名单

- `/app/auth/login` — 登录，无需 Token
- `/app/common/dicts` — 字典，无需 Token
- 其他 `/app/*` — 需要 HMAC Token（登录后获取，7天有效）
