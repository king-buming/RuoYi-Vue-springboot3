## 二、人脸录入流程

### 2.1 前提：上传人脸照片

用户先在「人员管理 → 人脸信息」为人员上传照片。

```
浏览器 [选择照片]
  │
  ▼
POST /common/upload  (MultipartFile)
  │
  ▼
CommonController.uploadFile()
  │  RuoYiConfig.getUploadPath() = "{project}/uploadPath/upload"
  │  FileUploadUtils.upload()
  │    → extractFilename() = "2026/06/05/face_12345.jpg"
  │    → file.transferTo("{project}/uploadPath/upload/2026/06/05/face_12345.jpg")
  │  → 返回 URL: "/profile/upload/2026/06/05/face_12345.jpg"
  │
  ▼
前端把 URL 写入 tb_worker_face.face_img_url

磁盘文件: .../RuoYi-Vue-springboot3/uploadPath/upload/2026/06/05/face_12345.jpg
数据库:   tb_worker_face.face_img_url = "/profile/upload/2026/06/05/face_12345.jpg"
```

### 2.2 人脸录入：提取特征写入底库

操作员在「AI算法仓 → 人脸注册」页面，对已上传照片的人员点击「录入」。

```
┌─ 前端 ─────────────────────────────────────────────────────┐
│ 点击 [录入] 按钮                                            │
│   → 检查 scope.row.faceImgUrl 是否存在                      │
│   → 弹出确认框："确认将[张三]的人脸录入到AI识别模型？"       │
│   → 用户确认                                               │
│   → POST /ai/face/register/1?modelCode=arcface_r100_001    │
└──────────────────┬──────────────────────────────────────────┘
                   │
┌──────────────────┴──────────────────────────────────────────┐
│ AiFaceRegisterController.registerFace(workerId=1,            │
│                                       modelCode="arcface_r100_001")
│                                                              │
│   try {                                                      │
│     faceRegisterService.registerFace(1, "arcface_r100_001"); │
│     return success("人脸录入成功");                           │
│   } catch (ServiceException e) {                             │
│     return error(e.getMessage()); // 如："人脸图片文件不存在" │
│   }                                                          │
└──────────────────┬──────────────────────────────────────────┘
                   │
┌──────────────────┴──────────────────────────────────────────┐
│ AiFaceRegisterServiceImpl.registerFace(workerId, modelCode)  │
│                                                              │
│ ① 校验 AI 模型                                              │
│   aiModelMapper.selectAiModelByCode("arcface_r100_001")     │
│   → AiModel { modelType="face_recognition",                  │
│               modelPath=".../ai-models/w600k_mbf.onnx" }    │
│   检查 model_type == "face_recognition" ✓                    │
│                                                              │
│ ② 检查重复录入                                              │
│   selectByWorkerIdAndModelCode(1, "arcface_r100_001")       │
│   → registerStatus != '1' → 可以录入 ✓                      │
│                                                              │
│ ③ 校验人员存在                                              │
│   workerMapper.selectTbWorkerById(1) → TbWorker ✓           │
│                                                              │
│ ④ 获取人脸照片 URL                                          │
│   workerFaceMapper.selectTbWorkerFaceList({workerId:1})     │
│   → faceImgUrl = "/profile/upload/2026/06/05/face_12345.jpg"│
│                                                              │
│ ⑤ 调用 AI 推理 ↓                                            │
└──────────────────┬──────────────────────────────────────────┘
                   │
┌──────────────────┴──────────────────────────────────────────┐
│ RealAiInferenceService.extractFaceFeature(                   │
│     modelCode="arcface_r100_001",                            │
│     faceImgUrl="/profile/upload/2026/06/05/face_12345.jpg") │
│                                                              │
│ ┌─ Step 1: 解析图片路径 ──────────────────────────────────┐ │
│ │ resolveImageFile("/profile/upload/2026/06/05/xxx.jpg")  │ │
│ │   profile = RuoYiConfig.getProfile()                    │ │
│ │          = ".../RuoYi-Vue-springboot3/uploadPath"       │ │
│ │   → File(".../uploadPath/upload/2026/06/05/xxx.jpg")    │ │
│ └─────────────────────────────────────────────────────────┘ │
│                                                              │
│ ┌─ Step 2: 加载 YOLOv12 人脸检测模型 ─────────────────────┐ │
│ │ getDetector()  // double-checked locking 懒加载          │ │
│ │   aiModelMapper.selectAiModelByCode("yolo_v8_obj_001")  │ │
│ │   → modelPath = ".../ai-models/yolov12m-face.onnx"      │ │
│ │   new OnnxFaceDetector(modelPath, 0.5, 0.45)            │ │
│ │     OrtEnvironment.getEnvironment()  // 全局单例         │ │
│ │     env.createSession("yolov12m-face.onnx")             │ │
│ └─────────────────────────────────────────────────────────┘ │
│                                                              │
│ ┌─ Step 3: YOLOv12 检测人脸 ─────────────────────────────┐ │
│ │ det.detect(faceFile)                                    │ │
│ │                                                          │ │
│ │ 3a. 读图                                                │ │
│ │   ImageUtils.readImageBGR(file)                         │ │
│ │   BufferedImage → getRGB() → BGR flat byte[]            │ │
│ │   像素索引: idx = (y*W + x)*3 + c                       │ │
│ │   c=0=B, c=1=G, c=2=R                                   │ │
│ │                                                          │ │
│ │ 3b. Letterbox 预处理                                    │ │
│ │   scale = min(640/W, 640/H)                             │ │
│ │   双线性插值缩放 → 居中 → 灰色(114)填充到 640×640       │ │
│ │   BGR保持不变 → normalize /255.0 → NCHW float[]         │ │
│ │   输入张量: (1, 3, 640, 640) float32                    │ │
│ │                                                          │ │
│ │ 3c. ONNX 推理                                           │ │
│ │   FloatBuffer.wrap(data) → OnnxTensor                   │ │
│ │   session.run() → 输出 (1, 300, 6)                      │ │
│ │   每个检测框: [x1, y1, x2, y2, conf, cls]               │ │
│ │                                                          │ │
│ │ 3d. 后处理                                              │ │
│ │   过滤 conf < 0.5 的框                                  │ │
│ │   Letterbox坐标 → 原图坐标: x = (x_lb - pad) / scale    │ │
│ │   贪心NMS: 按conf降序 → IoU>0.45抑制                    │ │
│ │   返回: List<float[]> faces = [[x1,y1,x2,y2,conf], ...] │ │
│ └─────────────────────────────────────────────────────────┘ │
│                                                              │
│ ┌─ Step 4: 裁切最大人脸 ──────────────────────────────────┐ │
│ │ 取 faces 中面积最大的 (w*h最大)                          │ │
│ │ cropFace(bgr, bestFace)                                 │ │
│ │   从 flat BGR 数组中提取人脸区域的子数组                 │ │
│ └─────────────────────────────────────────────────────────┘ │
│                                                              │
│ ┌─ Step 5: 加载 ArcFace 特征提取模型 ─────────────────────┐ │
│ │ getEmbedding("arcface_r100_001")                        │ │
│ │   aiModelMapper.selectAiModelByCode("arcface_r100_001") │ │
│ │   → modelPath = ".../ai-models/w600k_mbf.onnx"          │ │
│ │   new OnnxFaceEmbedding(modelPath)                      │ │
│ └─────────────────────────────────────────────────────────┘ │
│                                                              │
│ ┌─ Step 6: ArcFace 提取 512 维特征向量 ───────────────────┐ │
│ │ emb.extract(faceCrop, h, w)                             │ │
│ │                                                          │ │
│ │ 6a. 预处理                                              │ │
│ │   resizeBGR → 112×112 (双线性插值)                      │ │
│ │   BGR → RGB: 通道反转 [B,G,R] → [R,G,B]                │ │
│ │   归一化: (pixel - 127.5) / 127.5 → [-1, 1]            │ │
│ │   NCHW布局: float[3*112*112]                            │ │
│ │   输入张量: (1, 3, 112, 112) float32                    │ │
│ │                                                          │ │
│ │ 6b. ONNX 推理                                           │ │
│ │   session.run() → 输出 (1, 512)                         │ │
│ │   → float[512] embedding                                │ │
│ │                                                          │ │
│ │ 6c. L2 归一化                                           │ │
│ │   norm = sqrt(sum(e[i]^2))                              │ │
│ │   e[i] = e[i] / norm  (使 ||e|| = 1)                   │ │
│ │                                                          │ │
│ │   ★ 归一化后 dot(e1, e2) = 余弦相似度，值域 [-1, 1]     │ │
│ └─────────────────────────────────────────────────────────┘ │
│                                                              │
│ ┌─ Step 7: 序列化特征向量 ───────────────────────────────┐ │
│ │ float[512] → Float.floatToIntBits → byte[2048]          │ │
│ │ → Base64.getEncoder() → String                          │ │
│ │ → JSON: {"dim":512, "features":"<base64>"}              │ │
│ │ → AiInferenceResult.success(modelCode, json, elapsed)   │ │
│ └─────────────────────────────────────────────────────────┘ │
└──────────────────┬──────────────────────────────────────────┘
                   │
┌──────────────────┴──────────────────────────────────────────┐
│ 回到 AiFaceRegisterServiceImpl.registerFace()               │
│                                                              │
│ ⑥ 写入 ai_face_register                                     │
│   record.setFaceFeature('{"dim":512,"features":"<base64>"}')│
│   record.setRegisterStatus("1")  // 已录入                  │
│   record.setRegisterTime(new Date())                        │
│   faceRegisterMapper.insertAiFaceRegister(record)           │
│                                                              │
│ ⑦ 同步更新 tb_worker.face_status = '1'                      │
│   workerMapper.updateTbWorker(worker)                       │
│                                                              │
│   → return rows                                              │
└──────────────────┬──────────────────────────────────────────┘
                   │
┌──────────────────┴──────────────────────────────────────────┐
│ AiFaceRegisterController                                    │
│   → return success("人脸录入成功")                           │
│                                                              │
│ 前端收到 { code: 0, msg: "人脸录入成功" }                   │
│   → 弹窗提示 ✓                                              │
│   → getList() 刷新列表                                      │
│   → 该人员 AI录入状态 → "已录入" ✓                          │
└──────────────────────────────────────────────────────────────┘
```

### 2.3 AI 推理失败时（已修复 Bug #1）

```
特征提取失败（如：图片文件不存在、未检测到人脸、模型未加载）
  │
  ▼
AiInferenceResult.fail(modelCode, "人脸图片文件不存在: /profile/...")
  │
  ▼
AiFaceRegisterServiceImpl:
  ① 写入失败记录到 DB:
     register_status = '2'  (录入失败)
     fail_reason = "人脸图片文件不存在: /profile/..."
  ② throw new ServiceException("人脸图片文件不存在: ...")  ← 修复后抛异常
  │
  ▼
Controller catch (ServiceException e):
  → return error(e.getMessage())
  → 前端收到 { code: 500, msg: "人脸图片文件不存在: ..." }
  → 弹窗显示错误提示 (红色) ✓
  → 列表显示 "录入失败" + 失败原因 ✓
```

```
│ │                                                          │ │
│ │ OnnxFaceDetector.detect(tempFile)                        │ │
│ │   → Letterbox 640×640 → ONNX 推理 → NMS                 │ │
│ │   → List<float[]> faces  // 检测到的人脸列表             │ │
│ │                                                          │ │
│ │ 取最大人脸 → cropFace() →                               │ │
│ │                                                          │ │
│ │ OnnxFaceEmbedding.extract(faceCrop, h, w)                │ │
│ │   → resize 112×112 → BGR→RGB → [-1,1] → ONNX推理       │ │

│ └─────────────────────────────────────────────────────────┘ │
│                                                              │
│ ┌─ Step B: 1:N 底库比对 ──────────────────────────────────┐ │
│ │                                                          │ │
│ │ 从 DB 加载所有已录入的人脸特征:                          │ │
│ │ aiFaceRegisterMapper.selectFaceRegisterList(null, "1")   │ │
│ │   → List<AiFaceRegister>  // 所有 register_status='1'   │ │
│ │                                                          │ │
│ │ 遍历比对:                                                │ │
│ │ for (AiFaceRegister reg : registered) {                  │ │
│ │   // 从 JSON 反序列化特征向量                            │ │
│ │   float[] stored = parseFeatureFromJson(                 │ │
│ │       reg.getFaceFeature()  // {"dim":512,"features":".."}
│ │   );                                                     │ │
│ │   // Base64解码 → byte[2048] → Float.intBitsToFloat      │ │
│ │   // → float[512]                                        │ │
│ │                                                          │ │
│ │   // 计算余弦相似度                                       │ │
│ │   float sim = SimilarityUtils.cosine(stored, query);     │ │
│ │   // cos = dot(a,b)  (因为 a,b 已 L2 归一化, ||a||=||b||=1)
│ │                                                          │ │
│ │   if (sim > bestSim) {                                   │ │
│ │     bestSim = sim;                                       │ │
│ │     bestMatch = reg.getWorkerName();                     │ │
│ │     bestWorkerId = reg.getWorkerId();                    │ │
│ │   }                                                      │ │
│ │ }                                                        │ │
│ └─────────────────────────────────────────────────────────┘ │
│                                                              │
│ ┌─ Step C: 判断结果 ──────────────────────────────────────┐ │
│ │                                                          │ │
│ │ if (bestSim >= similarityThreshold (0.35)                │ │
│ │     && bestMatch != null) {                              │ │
│ │   → {"matched":true, "workerId":1, "personName":"张三",  │ │
│ │      "similarity":0.72, "confidence":0.72}               │ │
│ │ } else {                                                 │ │
│ │   → {"matched":false, "bestSimilarity":0.21}             │ │
│ │ }                                                        │ │
│ └─────────────────────────────────────────────────────────┘ │
└──────────────────┬──────────────────────────────────────────┘
                   │
┌──────────────────┴──────────────────────────────────────────┐
│ 回到 HwAttendanceServiceImpl.doCheckInValidation()           │
│                                                              │
│ if (!result.isMatched()) {                                   │
│   attendance.setCheckStatus("1");  // 失败                   │
│   attendance.setFailReason("人脸识别不匹配：" + errorMsg);   │
│   return;  // 打卡失败                                       │
│ }                                                            │
│                                                              │
│ // 人脸比对通过 → 继续 GPS 校验 → 写入打卡记录               │
│ attendance.setCheckStatus("0");  // 成功                     │
│ attendanceMapper.insertHwAttendance(attendance);             │
│                                                              │
│ → 返回 "打卡成功"                                            │
└──────────────────────────────────────────────────────────────┘
```

## 四、两个 ONNX 模型详解

### 4.1 YOLOv12 人脸检测 (`yolov12m-face.onnx`)

```
目的: 在一张图片中找出所有人脸的位置

输入:
  shape: (1, 3, 640, 640)
  dtype: float32
  格式:  NCHW (batch, channel, height, width)
  通道:  BGR (与 Python cv2.imread 一致)
  范围:  [0, 1]  (pixel / 255.0)

预处理 Pipeline:
  ┌──────────┐    ┌──────────┐    ┌───────────┐    ┌──────────┐
  │ 读取图片  │ → │ Letterbox│ → │ Normalize │ → │ HWC→NCHW │
  │ BGR byte[]│   │ 640×640  │   │  /255.0   │   │ float[]  │
  └──────────┘    └──────────┘    └───────────┘    └──────────┘
  BGR保持原样     居中+灰边填充    [0,1]范围        NCHW布局

输出:
  shape: (1, 300, 6)
  每行: [x1, y1, x2, y2, confidence, class_id]
  最多 300 个检测框

后处理:
  1. 过滤 conf < 0.5 的框
  2. 坐标从 letterbox 空间转回原图空间
  3. 贪心 NMS (IoU > 0.45 抑制)
  4. 返回保留下来的框 (按面积排序)
```

### 4.2 ArcFace 特征提取 (`w600k_mbf.onnx`)

```
目的: 将一张人脸图片转换为一个 512 维的"身份编码"向量
      同一人的不同照片 → 向量距离很近 (余弦相似度 > 0.5)
      不同人的照片     → 向量距离很远 (余弦相似度 < 0.2)

输入:
  shape: (1, 3, 112, 112)
  dtype: float32
  格式:  NCHW
  通道:  RGB (BGR→RGB 已转换)
  范围:  [-1, 1]  ((pixel - 127.5) / 127.5)

预处理 Pipeline:
  ┌──────────┐    ┌──────────┐    ┌───────────┐    ┌──────────┐
  │ Resize   │ → │ BGR→RGB  │ → │ Normalize │ → │ HWC→NCHW │
  │ 112×112  │   │ 通道反转  │   │ (p-127.5) │   │ float[]  │
  └──────────┘    └──────────┘   │   /127.5  │    └──────────┘
  不保持宽高比                    │ [-1,1]范围│

输出:
  shape: (1, 512)
  float[512] (已 L2 归一化, ||v|| = 1)

★ L2 归一化的意义 ★
  归一化后 dot(v1, v2) = cos(θ) = 余弦相似度
  值域 [-1, 1], 人脸特征通常 > 0
  阈值 0.35: 平衡准确率和召回率
  阈值 0.50: 高安全, 宁拒勿漏
  阈值 0.25: 宽松,  宁漏勿拒
```

## 五、特征向量存储格式

```
ai_face_register.face_feature (TEXT 字段)

存储:
  float[512] embedding
    ↓ Float.floatToIntBits() × 512
  byte[2048]  (大端字节序)
    ↓ Base64.getEncoder()
  String (2732 字符)
    ↓ 包装为 JSON
  {"dim":512, "features":"PwAAAD8AAAA/AAAAPwAA..."}

读取 (1:N 比对时):
  JSON.parse → "features" 字段
    ↓ Base64.getDecoder()
  byte[2048]
    ↓ Float.intBitsToFloat() × 512
  float[512] (L2 归一化后的向量)
```

## 六、关键配置参数

| 参数 | 默认值 | 位置 | 作用 |
|------|--------|------|------|
| `detection-code` | `yolo_v8_obj_001` | yaml → ai_model 表 | 人脸检测模型编码 |
| `conf-threshold` | `0.5` | yaml → OnnxFaceDetector | 检测置信度，低于此值丢弃 |
| `iou-threshold` | `0.45` | yaml → OnnxFaceDetector | NMS 重叠阈值，高于此值抑制 |
| `similarity-threshold` | `0.35` | yaml → doFaceRecognition | 人脸匹配阈值，高于此值认为同一人 |
| `ruoyi.profile` | 项目内 uploadPath/ | application.yml | 文件上传根路径 |
